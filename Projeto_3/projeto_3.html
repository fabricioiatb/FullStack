Introdução

C++ é uma linguagem de programação compilada multi-paradigma, incluindo suporte para programação orientada a objetos e genérica. Desde os anos 1990, é uma das linguagens comerciais mais populares, bastante usada na academia e no mercado.

Java é uma linguagem de programação orientada a objetos, desenvolvida na década de 90 na Sun Microsystems. Em 2008, foi adquirida pela Oracle Corporation. Java e C++ possuem muitas semelhanças estruturais.
